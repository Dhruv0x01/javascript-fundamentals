# JavaScript Fundamentals

This repository contains **topic-wise JavaScript examples** that I created while learning JavaScript fundamentals from tutorials and practice.

Each folder focuses on **one concept**, with a simple HTML / CSS / JavaScript demo to understand how that concept works in practice.

## Structure

topics/       # concept-wise examples (numbered by video)
projects/     # small projects built with those concepts

Each topic folder typically contains:
- index.html – basic markup
- style.css – minimal styling (when needed)
- script.js – JavaScript logic for the topic
